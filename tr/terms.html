Hizmet Şartları

1. Giriş

aBest.co web sitesine hoş geldiniz.

Bu Hizmet Şartları, https://www.abest.co web sitesine ve ilgili tüm içerik, özellik ve hizmetlere erişimi ve bunların kullanımını düzenler.

Bu web sitesine erişerek, bu Hizmet Şartlarını kabul etmiş olursunuz. Bu şartları kabul etmiyorsanız, bu web sitesini kullanmamalısınız.

2. Web Sitesi İşletmecisi

İşletmeci, bu Hizmet Şartlarını istediği zaman değiştirebilir. Değişiklikler bu sayfada yayınlanacaktır.

3. Web Sitesinin Amacı

aBest.co web sitesi, özellikle aşağıdaki alanlarda projeleri, fikirleri ve iş fırsatlarını sunmaya hizmet eder:

  • Gayrimenkul Geliştirme
  • Ticari Projeler
  • Enerji Projeleri
  • Altyapı
  • Turizm ve Otelcilik
  • İş Birlikleri ve Ortaklıklar

Bu web sitesinin içeriği yalnızca bilgilendirme amaçlıdır.

Web sitesi bağlayıcı teklifler teşkil etmez.

4. Web Sitesinin Kullanımı

Bu web sitesinin kullanımı genel olarak ücretsizdir.

Web sitesini yalnızca yasal amaçlarla kullanmayı ve aşağıdaki eylemleri gerçekleştirmemeyi kabul edersiniz:

  • Yürürlükteki yasaları ihlal etmek
  • Web sitesinin güvenliğini tehlikeye atmak
  • Web sitesinin altyapısını aşırı yüklemek
  • Kötü amaçlı yazılım yaymak

Özellikle şunlar yasaktır:

  • İzin almadan otomatik sistemler (botlar, kazıyıcılar) kullanmak
  • Web sitesi içeriğini kopyalamak veya dağıtmak
  • Güvenlik mekanizmalarını devre dışı bırakmak

5. Fikri Mülkiyet

Metinler, logolar, grafikler, resimler, düzen, tasarım ve yapı dahil olmak üzere bu web sitesindeki tüm içerik telif hakkı ile korunmaktadır.

İçerik, işletmecinin önceden yazılı izni olmaksızın kopyalanamaz, çoğaltılamaz, dağıtılamaz, yayınlanamaz veya ticari amaçlarla kullanılamaz.

aBest.co marka adı ve tüm ilgili içerikler işletmecinin mülkiyetindedir.

6. Proje Bilgileri

Bu web sitesinde gösterilen projeler, kavramlar ve iş fırsatları yalnızca bilgilendirme amaçlıdır. Bunlar bağlayıcı teklifler veya yatırım taahhütleri teşkil etmez. Tüm bilgiler her an değiştirilebilir.

7. İş Birliği ve Öneriler

Web sitesi, ziyaretçileri iş fikirleri, iş birlikleri veya öneriler sunmaya davet edebilir. Bir öneri sunarak, işletmecinin öneriyi inceleyebileceğini, bunu uygulama yükümlülüğü olmadığını ve ayrı bir anlaşma yapılmadıkça herhangi bir tazminat ödenmeyeceğini kabul etmiş olursunuz.

8. Sorumluluk Reddi

Bu web sitesinin içeriği mümkün olan en büyük özenle oluşturulmuştur. Ancak işletmeci, içeriğin doğruluğu, tamlığı, güncelliği veya kullanılabilirliği konusunda hiçbir sorumluluk kabul etmez. Kullanım riski size aittir.

9. Harici Bağlantılar

Bağlantı verilen harici web sitelerinin işletmecileri kendi içeriklerinden tek başlarına sorumludur.

10. Web Sitesinin Kullanılabilirliği

Kesintisiz kullanılabilirlik garanti edilemez. İşletmeci, web sitesini istediği zaman değiştirme veya geçici olarak devre dışı bırakma hakkını saklı tutar.

11. Gizlilik Politikası

Web sitesinin kullanımı ayrıca web sitesinde bulunan Gizlilik Politikamıza tabidir.

12. Uluslararası Kullanım

Bu web sitesi çeşitli ülkelerden (ABD, Almanya, Türkiye vb.) kullanıcılara yöneliktir. Kullanım, ilgili yürürlükteki yasalara uygundur.

13. Amerika Birleşik Devletleri'nde Kullanım

Amerika Birleşik Devletleri'ndeki kullanıcılar için, potansiyel gizlilik hakları (CCPA) dahil olmak üzere ABD yasalarının hükümleri ek olarak geçerlidir.

14. Sorumluluğun Sınırlandırılması

Yasaların izin verdiği ölçüde işletmeci; dolaylı zararlar, netice zararları, kar kaybı veya veri kaybından sorumlu değildir. Sorumluluk, kasıt veya ağır ihmal durumlarıyla sınırlıdır.

15. Hizmet Şartlarında Değişiklikler

İşletmeci, bu Hizmet Şartlarını istediği zaman değiştirebilir. Güncel sürüm bu web sitesinde mevcuttur.

16. Bölünebilirlik Maddesi

Herhangi bir hükmün geçersiz olduğu tespit edilirse, geri kalan hükümlerin geçerliliği etkilenmez.

17. Uygulanacak Hukuk

Bu Hizmet Şartları, ilgili yürürlükteki yasal hükümlere tabidir.
